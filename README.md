pyhabit
===============

HabitRPG python library (WIP)

Installation
------------

Install using pip

    pip install git+git://github.com/xeross/pyhabit

Requires
--------

* requests

Usage
-----

    from habitrpg import HabitAPI

    api = HabitAPI(<user_id>, <api_key>)
    api.user()
    ... # etc.
